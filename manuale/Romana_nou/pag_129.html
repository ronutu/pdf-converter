Alcătuieşte un enunț în care să foloseşti adjectivul pronominal posesiv sa .

  • Spirache ( însumi ) .... şi-a depus candidatura.
  • ( Niciun ) .... membru al familiei nu-i convine refuzul .... ( său ) inițial.
  • Se ştie …. ( care ) fiu nu-i vine să creadă că tatăl va înfrunta .... ( tot ) familia.
  • .... ( celălalt ) fete susțin …. ( acelaşi ) hotărâre a mamei.

Realizează acordul corect al adjectivului pronominal de întărire, scriind enunțurile în caiet: Ea ... a scris referatul. Mie ... (masculin) mi-a promis că va trimite cărțile mâine. Mie ... (feminin) mi-a dat florile. El ... a cercetat artefactele. Voi ... (masculin) ați fost de acord. Vouă ... (feminin) trebuia să vă spună adevărul. Ție ... (feminin) vreau să-ți trimit coletul. Ei ... au adus cărțile.
