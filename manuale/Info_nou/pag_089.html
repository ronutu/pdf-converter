Facilități ale mediului de dezvoltare pentru un limbaj

de programare: editare, rulare și depanare

Din experiența ta

1

Ai observat că textul unui program scris în C++ este colorat diferit. Ce fel de aplicație crezi că a fost folosită în scrierea programului?

Important

Pentru scrierea programelor într-un limbaj de programare, se folosește o aplicație specializată denumită mediu de dezvoltare . Un mediu de dezvoltare este un set de programe care ajută programatorul în scrierea programelor, oferindu-i acestuia uneltele necesare editării (scrierii) codului sursă, compilării , rulării și depanării programului. Prin compilare , un program este transformat din cod sursă în cod executabil care poate fi rulat de computer. Un mediu de dezvoltare gratuit, care poate fi rulat pe Windows, Linux și MacOS este Code::Blocks . Acesta poate fi descărcat de la adresa codeblocks.org . Poți folosi și aplicația online https://www.onlinegdb.com/online_c++_compiler pentru scrierea de programe în limbajul C++.

Descoperă și aplică!

2

Pentru a descărca și instala aplicația Code::Blocks pentru Windows, pe care o vom numi de acum înainte simplu CodeBlocks, efectuează următoarele operații:

  1. Deschide un browser și scrie în bara de adrese: codeblocks.org/downloads .
  2. Din pagina deschisă, alege Download the binary release .
  3. Din următoarea pagină deschisă, alege pachetul care conține în denumire mingw-setup.exe . În prezent, versiunea este codeblocks-20.03mingw-setup.exe .
  4. Descarcă fișierul și instalează aplicația.

3

Familiarizează-te cu interfața CodeBlocks, efectuând următoarele operații:

  1. Deschide aplicația, dând clic pe pictograma de pe desktop. ( Figura 7 )
  2. La prima rulare, alege compilatorul GNU GCC Compiler.
  3. După alegerea compilatorului, vei fi întrebat dacă dorești să fie deschise fișierele C/C++ cu CodeBlocks. Alege cum consideri că este mai bine pentru tine. De preferat, să alegi: Da.
  4. Pentru a explora toate facilitățile mediului de dezvoltare, începe prin a crea un proiect nou, dând clic pe Create a new project :.
  5. Din fereastra următoare ( New from template) , alege să creezi o aplicație de consolă ( Console application ) (Figura 8) , dând dublu clic sau un clic pe aceasta, apoi apasă butonul Go aflat în partea din dreapta sus a ferestrei.
  6. Se va deschide o nouă fereastră care te îndrumă în crearea unei aplicații. Poți ignora prima pagină. Există o opțiune pe care o poți bifa, ca să nu îți mai apară data viitoare această pagină.
