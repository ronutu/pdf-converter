Pentru a folosi anumite funcții, trebuie incluse bibliotecile care le conțin. Cel mai des, vei folosi biblioteca iostream , care conţine suport pentru a realiza operațiile de citire și de scriere și biblioteca cmath , care conține unele funcții utilizate în calcule matematice. Includerea lor se face ca în exemplul de mai jos: #include // include biblioteca iostream #include // include biblioteca cmath

În această zonă vei declara variabilele pe care le vei folosi în program. Poți declara variabile oriunde în program, dar ar fi indicat să fii ordonat(ă). Este obligatoriu să declari variabilele înainte de a începe să le utilizezi.

Pe măsură ce înveți mai multe despre declarații de funcții și exersezi, vei reuși să declari funcțiile proprii. Deocamdată, vei putea scrie un program corect care ar arăta astfel: #include // includ biblioteca iostream int a, b; // declar două variabile întregi int main() { int c; // am mai declarat o variabilă întreagă return 0; // aici se încheie programul, cu codul de eroare 0, adică totul e ok }
