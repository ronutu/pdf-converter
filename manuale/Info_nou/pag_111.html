Dacă răspunsul este afirmativ, afișează intervalul de etaje pe care le-au urcat împreună.

Exemplu:

Pentru x = 2, y = 7, a = 5, b = 10 se afișează 5-7, iar pentru x = 2, y = 7, a = 7, b = 10 se afișează „nu”.
